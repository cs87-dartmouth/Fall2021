#include <darts/factory.h>
template<typename T>
DartsFactory class

A specialization of the Factory for darts objects read from json.

Template parameters
T This factory will create objects of type shared_ptr<T>.

Contents

  • Reference
    • Base classes
    • Public static functions
    • Protected static functions

Additionally, DartsFactory allows storing previously parsed/created instances into a registry with register_instance() and retrieving them with their name using find().

Warning

If you want to use DartsFactory<T>::find() with more types T, you need to specialize it

Find a previously parsed/created SharedT, or create a new one.

Parameters
jp The json object for the parent
key Look for this json field in jp
Returns SharedT The newly created or retrieved object instance.

Return a SharedT pointer by parsing a JSON specification. If j is a string key: "name", then try to find a previously parsed SharedT with name "name". If j is a json object key: {}, then create a new SharedT with the specified parameters by calling create().
